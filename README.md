# 保育園候補マップ 静的サイト

## 使い方
- `index.html` をトップページとして公開してください。
- PCでは `pc.html`、iPhoneでは `mobile.html` を利用します。
- GitHub Pages / Cloudflare Pages / Netlify などに、このフォルダ内のファイルをまとめてアップロードすれば動きます。

## 画面
- PC版: 左に保育園リスト、右に地図。ピン・カードから公式WEB/Google Mapsに遷移。
- スマホ版: 上に軽量マップ、下に園カード。外部地図ライブラリ非依存で安定性優先。

## 注意
- PC版の地図表示はOpenStreetMapタイルを使います。
- スマホ版の地図は相対配置です。正確な経路確認は各カードのGoogle Mapsを使ってください。
- 一部の座標・0歳枠等は要確認です。正式な判断は江戸川区・各園公式情報で確認してください。
